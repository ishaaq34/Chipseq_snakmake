Consensus Peaks
===============

High-confidence reproducible peaks per condition, derived from IDR analysis.

These peaks passed the following filters:

1. Called by MACS3 in individual replicates (p-value < 0.01)
2. Reproducible across replicates (IDR score ≥ 1.3)
3. Merged across all pairwise comparisons

Consensus peaks represent the most reliable binding sites and are recommended for downstream analysis (motif discovery, differential binding, etc.).

Number of peaks and FRiP calculated on consensus set are reported.
